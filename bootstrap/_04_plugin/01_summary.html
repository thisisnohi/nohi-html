JavaScript 插件 概览

$(document).off('.data-api')解除以data-api 为命名空间并绑定在文档上的事件 $(document).off('.alert.data-api')针对某个特定的插件
$('#myModal').modal() // 以默认值初始化
$('#myModal').modal({ keyboard: false }) // initialized with no keyboard
$('#myModal').modal('show') // 初始化后立即调用 show 方法
通过页面元素获取插件的实例： $('[rel="popover"]').data('popover')
$.fn.modal.Constructor.DEFAULTS.keyboard = false // 将模态框插件的 `keyboard` 默认选参数置为 false
