工具提示

添加data-toggle="tooltip" 的属性来开启它的工具提示功能，然后指定data-placement="left" 工具提示的位置，可以是上下左右（top、bottom、left、right）。
选择性加入的功能出于性能方面的考虑，工具提示和弹框组件的 data 属性 api 是选择性加入的，也就是说你必须自己初始化他们。 
        <script type="text/javascript"> $(function() {
            $('.tooltip-demo').tooltip({
                selector: "[data-toggle=tooltip]",
                container: "body" })
            $("#testt").tooltip({})
            $('a').tooltip()
        }) </script>

            $().tooltip(options) ——为一组元素应用工具提示。
            .tooltip('show') ——展示工具提示。
            $('#element').tooltip('show')
            .tooltip('hide') ——隐藏工具提示。
            $('#element').tooltip('hide')
            .tooltip('toggle') ——展示或隐藏工具提示。
            $('#element').tooltip('toggle')
            .tooltip('destroy') ——隐藏并销毁工具提示。
            $('#element').tooltip('destroy')
